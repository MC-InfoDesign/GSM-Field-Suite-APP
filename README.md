## Installing an APK

Some of these apps aren't on the Play Store yet (beta phase). To install one manually:

1. Download the `.apk` from the links above
2. On your Android phone, allow "install from unknown sources" when prompted
3. Open the downloaded file, install, done

## Why no Play Store yet (for the beta apps)

These are still in active testing. Direct APK distribution lets me ship fixes fast and collect feedback before a public listing. If you hit a bug or have an idea, use the in-app report button, it opens a pre-filled email to contact@gsm-field-suite.com, or open a GitHub issue directly.
